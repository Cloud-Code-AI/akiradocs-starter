# Welcome to Akira Docs Starter

Akira is a powerful and flexible documentation template built on top of Nextra and inspired by modern documentation platforms. It's designed to help you create beautiful, functional, and easy-to-maintain documentation for your projects.

## Features of Akira Docs Starter

- **Easy to Use**: Get started quickly with our intuitive setup process.
- **Highly Customizable**: Tailor your docs to your needs using `akira.json`.
- **Beautiful Design**: Modern, clean interface inspired by top documentation platforms.
- **Powerful Search**: Built-in search functionality to help users find information quickly.
- **Responsive**: Looks great on devices of all sizes.
- **Dark Mode**: Easy on the eyes, day or night.

## Getting Started

1. Clone this repository
2. Install dependencies with `pnpm install`
3. Start the development server with `pnpm dev`
4. Start editing your docs in the `pages` directory

For more detailed instructions, check out our [Getting Started guide](./getting-started).

## Customization

Akira is designed to be easily customizable. Most settings can be adjusted in the `akira.json` file. For more information, see our [Configuration guide](./configuration).

## Based on Nextra

Akira is built on top of Nextra, a powerful site generation framework that leverages Next.js. To learn more about the underlying technology, visit the [Nextra documentation](https://nextra.site).

We hope Akira helps you create amazing documentation for your projects. Happy documenting!
